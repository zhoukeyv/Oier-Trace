# OIerDb 数据查询工具

一个简单的本地查询工具，用于查询OIer竞赛获奖信息。

## 功能特性

- 按比赛、奖项、分数查询
- 按省份、学校筛选
- 按年级范围筛选
- 按姓名简称查询
- 支持多条件组合查询

## 数据来源

本项目使用的数据来源于：

- [OIerDb-data-generator](https://github.com/OIerDb-ng/OIerDb-data-generator)
- [OIerDb-ng](https://github.com/OIerDb-ng)
